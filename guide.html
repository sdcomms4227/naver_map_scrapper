네이버 지도 검색 자동화 사용 가이드

기능 소개

  • 네이버 지도에서 키워드 기반 장소 검색
  • 검색 결과 필터링
  • 예약 가능 여부 확인
  • 검색 결과 JSON 파일로 저장

사용 방법

  1. 검색어 입력
    • 검색하고자 하는 키워드를 입력합니다 (예: "부산광역시 파티룸")
    • 지역명과 업종을 함께 입력하면 더 정확한 결과를 얻을 수 있습니다
  2. 필터 키워드 입력 (선택사항)
    • 검색 결과를 필터링할 키워드를 입력합니다
    • 여러 키워드는 쉼표(,)로 구분합니다
    • 예: "파티룸, 대관"
  3. 실행 모드 선택
    • headless: 백그라운드에서 실행 (브라우저 창이 보이지 않음)
    • visible: 브라우저 창이 보이는 상태로 실행

주의사항

  • 검색 결과가 많을 경우 시간이 오래 걸릴 수 있습니다
  • 네트워크 상태에 따라 검색 속도가 달라질 수 있습니다
  • 브라우저 자동화 특성상 간혹 오류가 발생할 수 있습니다
  • 검색 중 브라우저를 직접 조작하지 마세요

팁

  • 정확한 검색을 위해 구체적인 지역명을 포함하세요
  • 필터 키워드는 검색 결과의 카테고리나 태그를 기준으로 합니다
  • 검색 중 오류가 발생하면 잠시 후 다시 시도해보세요
  • 검색 결과는 자동으로 JSON 파일로 저장됩니다

검색 결과 형식

{
    "id": 1,
    "name": "장소명",
    "filter": "카테고리",
    "address": "주소",
    "tel": "전화번호",
    "isBookable": true/false,
    "bookingItemsCount": 0
}

제한사항

  • 네이버 지도의 검색 결과 페이지 구조가 변경되면 동작하지 않을 수 있습니다
  • 한 번에 너무 많은 검색을 실행하면 네이버에서 일시적으로 차단될 수 있습니다
  • 예약 정보는 네이버 예약 시스템을 사용하는 장소만 확인 가능합니다
  • 웹 브라우저 환경에 따라 일부 기능이 제한될 수 있습니다

문제 해결

  • 검색이 시작되지 않는 경우
    • 검색어가 올바르게 입력되었는지 확인하세요
    • 브라우저를 새로고침한 후 다시 시도해보세요
  • 검색 중 오류가 발생하는 경우
    • 중지 버튼을 눌러 현재 검색을 중단하세요
    • 잠시 후 다시 시도해보세요
    • 필터 키워드를 조정해보세요
